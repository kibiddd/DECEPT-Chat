This dataset is part of the DECEPT dataset.

It contains scam and non-scam real-life chat segments on Instant Messaging platforms.

This dataset is part of the contributions of the paper under review in the Annals of Telecommunication, "Catch me if you can: small language models against scams in instant messaging", an extended version of our paper "Chat or Trap? Detecting Scams with Large Language Models", published in CSNet 2024.
